## Want to contribute?

Here's how to setup the module dev environment:

```bash
# clone this repo
git clone https://github.com/danielesteban/volumemarcher.git
cd volumemarcher
# install dependencies
npm install
# start the environment:
npm start
# open http://localhost:8080/example in your browser
```
